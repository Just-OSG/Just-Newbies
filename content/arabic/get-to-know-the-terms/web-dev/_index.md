title: تطوير الويب
weight: 6

تطوير الويب هو عملية بناء websites وتطبيقات ال web. ينقسم إلى frontend (ما يراه المستخدمون ويتفاعلون معه) و backend (منطق ال server وال databases). مع HTML و CSS و JavaScript كتقنيات أساسية، تطور تطوير الويب ليشمل frameworks وأدوات قوية تجعل إنشاء websites حديثة وتفاعلية أسهل من أي وقت مضى. إنها واحدة من أكثر نقاط الدخول إلى ال programming سهولة.

**المواضيع:** Frontend مقابل Backend، أساسيات HTML/CSS/JavaScript، Frameworks الحديثة، Responsive design، أدوات ال Browser dev

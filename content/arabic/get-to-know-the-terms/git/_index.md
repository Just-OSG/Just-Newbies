---
title: Version Control و Git
weight: 1
---

ال Version control هو نظام يتتبع التغييرات في الملفات عبر الزمن، مما يسمح لك باستدعاء إصدارات معينة لاحقاً. Git هو نظام ال version control الأكثر شعبية يستخدمه المطورون حول العالم. يساعد الفرق على التعاون في ال code، وتتبع التغييرات، وإدارة إصدارات مختلفة من ال projects. فهم Git ومنصات مثل GitHub أساسي لأي مطور، حيث يُستخدم في كل بيئة تطوير برمجيات تقريباً.

**المواضيع:** أساسيات ال Version control، Git workflow، Branches والدمج، التعاون في GitHub، أوامر Git الأساسية

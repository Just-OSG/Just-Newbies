title: أنظمة التشغيل
weight: 9
---

نظام التشغيل (Operating System - OS) هو ال software الذي يدير موارد ال hardware وال software في الحاسوب ويوفر خدمات لل programs. Windows و macOS و Linux هي أنظمة التشغيل الأكثر شيوعاً. فهم كيفية عمل أنظمة التشغيل، وكيفية التنقل فيها باستخدام ال command line، ومعرفة اختلافاتها يساعدك على العمل بكفاءة أكبر وحل المشكلات بفعالية.

**المواضيع:** ما هو ال OS، Windows/macOS/Linux، File systems، أساسيات ال Command line، Process management

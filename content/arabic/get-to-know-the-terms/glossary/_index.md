مجموعة من مصطلحات ال IT الشائعة والتعريفات لمساعدتك على التنقل في عالم التكنولوجيا.

**المواضيع:** مصطلحات IT الأساسية، الاختصارات الشائعة، المفاهيم الرئيسية

---

## A

**API (Application Programming Interface)** - مجموعة من القواعد وال protocols التي تسمح لتطبيقات ال software المختلفة بالتواصل مع بعضها البعض.

**Algorithm** - إجراء أو صيغة خطوة بخطوة لحل مشكلة أو إكمال مهمة.

## B

**Backend** - الجانب الخلفي (server-side) من تطبيق الويب الذي يتعامل مع معالجة ال data وال storage ومنطق ال business.

**Bug** - خطأ أو عيب في ال software يتسبب في إنتاج نتائج غير صحيحة أو غير متوقعة.

## C

**Cloud Computing** - تقديم خدمات الحوسبة (servers، storage، databases، networking) عبر الإنترنت.

**Commit** - لقطة من التغييرات التي تم إجراؤها على الملفات في نظام version control مثل Git.

## D

**Database** - مجموعة منظمة من ال data المنظمة المخزنة إلكترونياً في نظام حاسوب.

**Debugging** - عملية العثور على الأخطاء أو ال bugs في ال software code وإصلاحها.

## F

**Framework** - مجموعة قابلة لإعادة الاستخدام من ال libraries وال tools التي توفر بنية لبناء تطبيقات ال software.

**Frontend** - الجانب الأمامي (client-side) من تطبيق الويب الذي يتفاعل معه المستخدمون مباشرة (HTML، CSS، JavaScript).

## I

**IDE (Integrated Development Environment)** - تطبيق software يوفر أدوات شاملة لتطوير ال software في مكان واحد.

**IP Address** - تسمية رقمية فريدة يتم تعيينها لكل جهاز متصل بشبكة حاسوب.

## V

**Variable** - حاوية مسماة تخزن قيم ال data في البرمجة.

**Version Control** - عملية تتبع وإدارة التغييرات في ال software code عبر الزمن.

**Virtual Machine** - محاكاة software لحاسوب فعلي يشغل نظام تشغيل وتطبيقات.

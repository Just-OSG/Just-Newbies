title: Cloud Computing
weight: 3
---

ال Cloud computing يشير إلى تقديم خدمات الحوسبة - بما في ذلك ال servers وال storage وال databases وال networking وال software وأكثر - عبر الإنترنت. بدلاً من امتلاك servers فعلية و infrastructure، يمكن للشركات والأفراد استئجار موارد الحوسبة من مزودي ال cloud مثل AWS و Microsoft Azure و Google Cloud. هذا أحدث ثورة في كيفية بناء ال applications ونشرها وتوسيع نطاقها في ال IT الحديثة.

**المواضيع:** ما هو ال Cloud computing، نماذج الخدمة (IaaS/PaaS/SaaS)، مزودو ال Cloud الرئيسيون، Cloud storage، Virtual machines
